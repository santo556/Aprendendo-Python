	Operadores Aritméticos
==> + Adição
==> - Subtração
==> * Multiplicação
==> / Divisão
==> ** Potência
==> // Divisão Inteira
==> % Resto da Divisão
-------------------------------------------------------------------------------------------------------------------------------
	1 - EXEMPO:
		==> 5 + 2 == 7
		==> 5 - 2 == 5
		==> 5 * 2 == 10
		==> 5 / 2 == 2.5
		==> 5 ** 2 == 25
		==> 5 // 2 == 2
		==> 5 % 2 == 1
------------------------------------------------------------------------------------------------------------------------------
			Ordem de Precedêndia
==> 1° => ()
==> 2° => **
==> 3° => * / // %
==> 4° => + -
------------------------------------------------------------------------------------------------------------------------------

	2 - EXEMPLO:
==> 5 + 3 * 2 == 11
==> 3 * 5 + 4 ** 2 == 31
==> 3 * (5 + 4 ) ** 2 == 243
------------------------------------------------------------------------------------------------------------------------------
			Prática

{:>20} ==> alinhado a direita
{:<20} ==> alinhado a esquerda
{:^20} ==> centralizado
------------------------------------------------------------------------------------------------------------------
n1 = int(input('Digite um valor: '))
n2 = int(input('Outro valor: '))
s = n1 + n2
m = n1 * n2
d = n1 / n2
di = n1 // n2
e = n1 ** n2
print('A soma é {} e o produto é {} e a divisão é {:.3f} '.format(s, m, d))
print('Divisão inteira é {} e a potência é {} '.format(di, e))
